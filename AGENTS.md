# Billi Bot Repository Guide for AI Agents

## Overview
The billi repository is the core open-source framework of the Billi trading system.

## Key Characteristics

### Technology Stack
- **Python** - Primary language
- **FastAPI** - API framework
- **NumPy** - Array operations

## File Structure
- `billi/` - Main source code
- `tests/` - Test suite

## Commands
```bash
billi run
```
